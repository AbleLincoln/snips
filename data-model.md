## Snippet
| property    | type                            |
| ----------- | ------------------------------- |
| id          | string (generated by `shortid`) |
| author      | User                            |
| code        | string                          |
| title       | string                          |
| description | string                          |
| language    | Language                        |
| comments    | [Comment]                       |
| favorites   | number                          |

## User

| property    | type              |
| ----------- | ----------------- |
| username    | string _(unique)_ |
| name        | string            |
| profile pic | file _(image)_    |
| password    | string _(hashed)_ |
| posts       | [Snippet]         |
| comments    | [Comment]         |

## Language

| property | type              |
| -------- | ----------------- |
| name     | string _(unique)_ |

## Comment

| property | type    |
| -------- | ------- |
| id       | string  |
| body     | string  |
| author   | User    |
| snippet  | Snippet |
